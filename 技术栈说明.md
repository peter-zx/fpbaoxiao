# 报销费用填写工具 — 技术栈说明

> 本文档用于团队深入学习与二次开发。涵盖项目架构、核心模块、技术选型细节与关键实现原理。

---

## 一、项目概览

| 项 | 说明 |
|---|---|
| **项目名称** | 报销费用填写工具（baoxiao） |
| **定位** | 辅助报销人员快速整理费用图片、填写报销单、导出标准 Excel 的桌面工具 |
| **技术路线** | Python HTTP Server + HTML/CSS/JS 前端 + xlsxwriter（跨平台优先）/ COM/Office Excel（桌面备选） |
| **运行方式** | 浏览器访问本地服务（localhost:8765），xlsxwriter 直接生成 xlsx 文件 |
| **打包分发** | PyInstaller 单 exe（Windows），可复制到任何 Windows 电脑独立运行 |
| **部署方式** | 也可作为云端服务部署到服务器（Flask + Nginx） |

### 5.2 API 路由

|---|---|---|
| `GET` | `/` | 返回 index.html |
| `GET` | `/api/data` | 读取 data.json |

### 5.3 Excel 生成引擎（两层降级）

这是本项目最核心的技术部分。按优先级尝试：

```
第1优先：xlsxwriter（当前生产方案）
         ├─ 优点：纯 Python，跨平台，稳定，不依赖本地 Office
         ├─ 优点：insert_image() + object_position=2 → 真正的"放置在单元格中"
         ├─ 缺点：需要 Excel 365 2023+ 或新版 WPS 支持 Place-in-Cell
         └─ 适用：所有 Windows/Linux/macOS 环境
            ↓ xlsxwriter 不可用时
第2优先：win32com COM（Microsoft Excel / WPS）
         ├─ 优点：原生 Excel，完全控制，兼容所有 Excel 版本
         ├─ 缺点：需要本机安装 Office，Windows 限定
         └─ 适用：桌面版回退方案
```

**xlsxwriter 图片嵌入原理：**
- `ws.insert_image(row, col, img_path, {'object_position': 2})`
- `object_position=2` = 随单元格移动和调整大小 = Excel "放置在单元格中"
- 图片按行高/列宽等比缩放，自动适应单元格尺寸

### 5.4 win32com COM 导出流程（回退方案）

```python
# 1. 启动 Excel COM
excel = win32com.client.Dispatch('Excel.Application')
wb = excel.Workbooks.Add()
ws = wb.Worksheets(1)

# 2. 设置列宽（独立 try-except，单列失败不影响其他）
col_widths = [12, 16, 14, 18, 12, 40, 8, 14]  # 第6列=40（图片列）
for col_idx, width in enumerate(col_widths, 1):
    try:
        ws.Columns(col_idx).ColumnWidth = width
    except Exception as e:
        logging.warning(f"设置列宽失败: {e}")  # 继续，不中断

# 3. 写入标题 + 设置标题行样式
ws.Rows(1).RowHeight = 40
for col_idx, header in enumerate(headers, 1):
    cell = ws.Cells(1, col_idx)
    cell.Value = header
    cell.Font.Bold = True
    cell.Font.Color = 0x000000       # 黑色
    cell.Interior.Color = 0xFFFFFF  # 无背景色
    cell.HorizontalAlignment = -4108  # xlCenter
    cell.VerticalAlignment = -4160   # xlVAlignCenter

# 4. 写入数据行（先赋值，后设置样式，避免被覆盖）
for row_idx, record in enumerate(records, 2):
    # 设置行高=300（图片列高度）
    ws.Rows(row_idx).RowHeight = 300
    
    # 先赋值
    ws.Cells(row_idx, 1).Value = record['time']
    # 再设置样式（避免值赋值覆盖对齐）
    style_cell(ws.Cells(row_idx, 1))
    
    # ... 其他列同理 ...

# 5. 插入图片（匹配单元格尺寸，居中显示）
if record.get('imageIndex'):
    pil_img = parse_base64_image(record['image'])
    orig_w, orig_h = pil_img.size  # 原始像素尺寸
    
    cell = ws.Cells(row_idx, 6)
    cell_width = cell.Width   # 单元格宽度（磅）
    cell_height = cell.Height  # 单元格高度（磅）
    
    # 等比缩放：图片适应单元格（不放大）
    scale = min(cell_width / orig_w, cell_height / orig_h)
    scale = min(scale, 1.0)  # 不放大，只缩小
    
    display_w = orig_w * scale
    display_h = orig_h * scale
    offset_x = (cell_width - display_w) / 2  # 水平居中
    offset_y = (cell_height - display_h) / 2  # 垂直居中
    
    pic = ws.Shapes.AddPicture(
        img_path,
        LinkToFile=False,       # 不链接原文件
        SaveWithDocument=True,  # 嵌入 xlsx
        Left=cell.Left + offset_x,
        Top=cell.Top + offset_y,
        Width=display_w,
        Height=display_h
    )
    pic.Placement = 1  # 随单元格移动和缩放

# 6. 合计行
total_row = len(records) + 2
ws.Cells(total_row, 4).Value = '合计：'
ws.Cells(total_row, 5).Value = total
ws.Cells(total_row, 5).NumberFormat = '¥#,##0.00'

# 7. 保存（临时文件 → 前端下载）
tmp_path = f'exports/baoxiao_export_{timestamp}.xlsx'
wb.SaveAs(os.path.abspath(tmp_path))
wb.Close()
excel.Quit()
```

### 5.5 win32com 已知问题与处理

| 问题 | 原因 | 处理方式 |
|---|---|---|
| `不能设置类 Range 的 RowHeight 属性` | 某些状态下行列属性不可写 | 独立 try-except，失败继续 |
| `不能设置类 Range 的 ColumnWidth 属性` | 合并单元格导致 | 同上，单列失败不影响其他 |
| 回退到 xlsxwriter | COM 调用失败 | 保留 xlsxwriter 作为降级方案 |
| 图片超出单元格 | 原始图片尺寸大于单元格 | 用 `cell.Width/Height` 计算缩放 |

### 5.6 数据存储

**路径：** `data.json`（与 server_cloud.py 同目录）

**加载逻辑：**
```python
def load_data():
    if os.path.exists('data.json'):
        with open('data.json', 'r', encoding='utf-8') as f:
            return json.load(f)
    return {'expense': [], 'reimburse': []}
```

**保存逻辑：**
```python
def save_data(data):
    # 读取旧数据，保留 _checked 状态
    old = load_data()
    for template in ['expense', 'reimburse']:
        old_ids = {r['id']: r for r in old.get(template, [])}
        for r in data.get(template, []):
            if r['id'] in old_ids and '_checked' in old_ids[r['id']]:
                r['_checked'] = old_ids[r['id']]['_checked']
    
    with open('data.json', 'w', encoding='utf-8') as f:
        json.dump(data, f, ensure_ascii=False, indent=2)
```

## 六、关键模块详解

### 6.1 单实例锁机制

```python
def acquire_lock():
    lock_file = 'server.lock'
    if os.path.exists(lock_file):
        try:
            with open(lock_file) as f:
                old_pid = int(f.read().strip())
            if psutil.pid_exists(old_pid):
                print(f"已有实例运行（PID {old_pid}），退出。")
                sys.exit(0)
        except: pass
    
    with open(lock_file, 'w') as f:
        f.write(str(os.getpid()))
```

### 6.2 端口冲突自动检测

```python
def find_free_port(start=8765, end=8781):
    for port in range(start, end):
        try:
            s = socket.socket()
            s.bind(('127.0.0.1', port))
            s.close()
            return port
        except OSError:
            continue
    raise Exception("无可用端口")
```

### 6.3 图片 Base64 处理

```python
def parse_base64_image(base64_str):
    """从 base64 字符串解析并保存为 PNG 文件"""
    if not base64_str or not base64_str.startswith('data:image'):
        return None
    header, data = base64_str.split(',', 1)
    img_bytes = base64.b64decode(data)
    img = PILImage.open(io.BytesIO(img_bytes))
    return img
```

### 6.4 中文文件名下载（避免乱码）

**问题：** 直接 `Content-Disposition: attachment; filename="报销单.xlsx"` 在不同浏览器编码不一致。

**解决方案：** 前端用 Blob + URL.createObjectURL 下载：
```javascript
const blob = await response.blob();
const url = URL.createObjectURL(blob);
const a = document.createElement('a');
a.href = url;
a.download = decodeURIComponent(filename);
a.click();
URL.revokeObjectURL(url);
```
